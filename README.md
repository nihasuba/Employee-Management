# Employee Management System

A modern and responsive **Full-Stack Employee Management Application** built to manage employee details seamlessly with full CRUD operations.

# 🛠️ Tech Stack

## 🎨 Frontend
- ⚛️ React.js
- 🎨 Tailwind CSS


## ⚙️ Backend
- 🚀 NodeJS


## 🛢️ Database
- 🍃 MongoDB (Atlas)

## ☁️ Deployment
- ▲ Vercel (Frontend & Backend)

---

# ⚙️ Backend Setup

## 1️⃣ Navigate to Server

```bash
cd backend
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 3️⃣ Create Environment File

Create a `.env` file inside the `server` directory.

```env
MONGO_URI="your MONGO DB URL"

```

# 💻 Frontend Setup

## 1️⃣ Navigate to Client

```bash
cd ../frontend
```

## 2️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm run dev
```
